    * `cost_100`: Tarifa +100KGS

### 2. Script de Ingesta y Limpieza (`import_rates_air.py`)
Escribe un script robusto utilizando `pandas` para leer el CSV y realizar la carga masiva.

**Reglas Críticas de Procesamiento:**

1.  **Lectura del Archivo:**
    * Nombre: `prueba tarifas AEREO IMPORT AIR.csv`
    * Delimitador: Punto y coma (`;`).
    * **Codificación:** Intenta `utf-8` o `latin-1` si es necesario.

2.  **Lógica de Limpieza de Datos (Muy Importante):**
    * **Relleno de Datos (Forward Fill):** Las columnas `Airport of destination` y `airport of origin` tienen celdas vacías (NaN) que implican que se repite el valor de la fila superior. Debes aplicar `.ffill()` en estas columnas inmediatamente después de cargar el dataframe.
    * **Limpieza de Costos:** Las columnas de peso (`+45KGS`, etc.) contienen:
        * Caracteres invisibles o espacios (ej: `9.32 `). Usa `.str.strip()`.
        * El caracter `/` que indica "No aplica". Conviértelo a `None` o `NaN`.
        * Conviértelos a tipo `Float`.
    * **Textos:** Normaliza puertos y aerolíneas (`.strip().upper()`).

3.  **Mapeo de Columnas:**
    Mapea las columnas del CSV a la base de datos de la siguiente forma:
    * `Airport of destination` -> `pod_name`
    * `airport of origin` -> `pol_name`
    * `CARRIER` -> `carrier_name`
    * `ROUTING` -> `routing`
    * `Transit time` -> `transit_time`
    * `Flight schedule` -> `frequency`
    * `PACKAGING ACCEPTED` -> `packaging_type`
    * `+45KGS` -> `cost_45`
    * `+100KGS` -> `cost_100`
    * ... (y así sucesivamente hasta +1000KGS)

4.  **Constantes:**
    * `transport_type` = 'AEREO'
    * `currency` = 'USD'
    * `validity_date` = '2025-12-31' (o define una constante configurable al inicio del script).

5.  **Persistencia:** Simula la inserción o imprime los registros listos para insertar.
