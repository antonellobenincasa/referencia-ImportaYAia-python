* `id`: Integer, PK.
* `transport_type`: String (Hardcoded: 'AEREO').

* `pod_name`: String (Aeropuerto de Destino - columna 'Airport of destination').

* `transit_time`: String (Tiempo de tránsito).

    * `cost_45`: Float (Tarifa +45KGS).

    * `cost_300`: Float (Tarifa +300KGS).
    * `cost_500`: Float (Tarifa +500KGS).
    * `cost_1000`: Float (Tarifa +1000KGS).
* **Campos calculados/por defecto:**
    * `currency`: String (Valor por defecto: 'USD', ya que el archivo no lo especifica).
    * `validity_date`: Date (Valor por defecto: '2025-12-31', ya que el archivo no trae vigencia).

### 2. Script de Ingesta (`import_rates_air.py`)
Escribe un script en Python utilizando `pandas` que cumpla con las siguientes reglas de negocio estrictas para limpiar los datos "sucios" del archivo:

**Reglas de Limpieza y Transformación:**

1.  **Lectura del Archivo:**
    * Usa delimitador `sep=';'`.
    * El archivo contiene filas donde el Origen y Destino están vacíos (NaN) porque pertenecen a la agrupación de la fila anterior. **Debes aplicar `df.ffill()` (Forward Fill)** en las columnas `Airport of destination` y `airport of origin` inmediatamente después de cargar los datos para corregir esto.

2.  **Limpieza de Precios (Crítico):**
    Las columnas de peso (`+45KGS` a `+1000KGS`) contienen datos sucios como:
    * El caracter `/` que significa "No aplica". Transfórmalo a `None` o `NaN`.
    * Espacios en blanco no estandar (non-breaking spaces `\xa0`). Debes hacer `.astype(str).str.strip()` antes de convertir.
    * Convierte finalmente a `Float`.

3.  **Mapeo de Columnas:**
    * `Airport of destination` -> `pod_name`
    * `airport of origin` -> `pol_name`
    * `CARRIER` -> `carrier_name`
    * `ROUTING` -> `routing`
    * `Transit time` -> `transit_time`
    * `Flight schedule` -> `frequency`
    * `PACKAGING ACCEPTED` -> `packaging_type`

4.  **Normalización de Texto:**
    * Asegura que los códigos de aeropuerto y carrier estén limpios: `.str.strip().str.upper()`.
